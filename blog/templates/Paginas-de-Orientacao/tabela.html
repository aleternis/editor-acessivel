Tabelas

Para pessoas sem problemas de visão, as informações contidas em tabelas são compreendidas apenas avaliando o conteúdo da tabela. Dados específicos são encontrados cruzando visualmente colunas e linhas. No entanto para pessoas portadoras de deficiência visual, a compreensão e obtenção de dados de uma tabela não é uma tarefa fácil se essa tabela for construída de forma não acessível. Pessoas portadoras de deficiência visual utilizam-se geralmente de leitores de tela. Os leitores de tela não lêem a "tela" e sim decodificam a sua estrutura HTML.

Importante:

  • A finalidade da tabela é apresentar dados tabulares. Não utilize tabelas para diagramar e/ou formatar conteúdo, pois esta prática causa problemas não só a acessibilidade, mas também de performance no carregamento da página.
  • Mesmo com todos os elementos que proporcionem a acessibilidade, tabelas complexas são de difícil compreensão para o usuário que utiliza leitores de tela. Sempre que possível devem ser usadas tabelas simples. Quando possível, as tabelas complexas devem ser divididas em tabelas simples.

Ao criar uma tabela, é fundamental considerar os seguintes elementos:

Descrição da tabela

Item fundamental para ajudar o usuário a interpretar a tabela. Na descrição são inseridas informações relacionadas a sua finalidade primária e estrutura geral.

Cabeçalhos da Tabela

Usuários sem problemas de visão, em geral, fazem a leitura dos cabeçalhos no topo de uma coluna e no início de uma linha para identificar o conteúdo de uma determinada célula da tabela. Uma tarefa bem simples que requer apenas a identificação de cabeçalhos e de dados contidos na tabela.
